Rodar via Terminal

Use o comando abaixo no terminal para rodar o scraper com as configurações atuais:

python -m core.scraper.cli --url "{{ projeto.url_booking }}" --id "{{ projeto.id }}" --ano "{{ projeto.ano_referencia }}"
