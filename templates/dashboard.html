Como rodar o Scraper

Copie e cole os comandos abaixo no terminal para atualizar os dados de mercado deste projeto.

1. Entrar na pasta do projeto 
cd /caminho/do/projeto
2. Rodar o scraper (substitui dados anteriores automaticamente) 
python -m core.scraper.cli --url "{{ projeto.url_booking }}" --id "{{ projeto.id }}" --ano "{{ projeto.ano_referencia }}"

⚠️ O scraper apaga e recria automaticamente os dados anteriores ao concluir com sucesso.
